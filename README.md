# Kaggle Playground Series - Season 6, Episode 6: Predicting Stellar Class

## Problem Description
The goal of this competition is to predict the stellar class of celestial objects based on astronomical data. The target classes are:
- GALAXY
- STAR
- QSO (Quasar)

## Project Structure
- `data/`: Contains the competition data.
- `notebooks/`: Exploratory Data Analysis (EDA) and experimentation.
- `src/`: Source code for data processing, feature engineering, and modeling.
- `requirements.txt`: Project dependencies.
